猎豹浏览器皮肤定制文档
====================

关于此文档
--------------------

本文档是猎豹浏览器官方皮肤定制文档，目的是帮助用户快速找到皮肤配置文件中配置项的详解，以及简单的皮肤制作教程。

包含的主要内容为 `mainfest.json` 、`layout_main` 和 `layout_parameter` 等配置文件中配置项的具体解释，以及如何制作一款猎豹浏览器专属皮肤的简易教程。
